Tour

  • Create an Escrow Agreement
  • Respond to an Escrow Request

Create the Contract

Enter the details of the agreement into Eskonto. You should be careful to enter all the deatils because if their is a dispute we will use these details as part of dispute resolution.

When you create an escrow agreement with Eskonto we generate two cryptographic invite codes. One for you and one for the buyer/seller.

You never to see the other persons invite code, we store that safely on our servers.

Send the Contract

Now, whoever is the Payee in the transaction generates a payment invitation with Eskonto using our code that runs in the browser.

When the other party receives the inviation they can accept it and a Bitcoin address will be generated for the escrow.

We never see the payment invitaion code (you have to email it yourself) and therefore as an escrow provider we are unable to access the funds without help from the payer or the payee.

Wait for Funds

Once funds have been received buy the bitcoin address Eskonto allows the buyer to release the funds or the seller to refund the Bitcoins. Only if there is a dispute between the parties do we get involved. At this pojnt we can use our knowledge of the original 2 invite codes to side with either the buyer or the seller.
